Module 1: Variables in Python

Dynamic Typing

Variables do not need to be declared with any particular type, and can even change type after they have been set.

Example: Dynamic Typing

x = 4 # x is of type int
x = "Sally" # x is now of type str
print(x)

Checking Data Types

You can get the data type of a variable with the type() function.

Example: Using type()

x = 5
y = "John"
print(type(x))
print(type(y))

String Declarations

String variables can be declared either by using single or double quotes.

Example: String Quotes

x = "John" # is the same as
x = 'John'

Case Sensitivity

Variable names are case-sensitive.

Example: Case Sensitivity

a = 4
A = "Sally" # A will not overwrite a

2. Variable Names: Rules and Conventions

Variable names can be short or descriptive. They must follow specific rules:

  • Start with a letter or underscore.
  • Cannot start with a number.
  • Contain only alphanumeric characters and underscores.
  • Are case-sensitive.

Legal and Illegal Variable Names

Legal: myvar, my_var, _my_var, myVar, MYVAR, myvar2
Illegal: 2myvar, my-var, my var

Multi-Word Variable Names

Use conventions for readability:

  • Camel Case: myVariableName
  • Pascal Case: MyVariableName
  • Snake Case: my_variable_name

Python User Input: input() and raw_input()

Python allows user input. Use input() in Python 3.6+ and raw_input() in Python 2.7.

Example: User Input

Python 3.6+: username = input("Enter username:"); print("Username is: " + username)
Python 2.7: username = raw_input("Enter username:"); print("Username is: " + username)

Assigning Multiple Values

Many values to multiple variables: x, y, z = "Orange", "Banana", "Cherry"
One value to multiple variables: x = y = z = "Orange"
Unpacking a collection: fruits = ["apple", "banana", "cherry"]; x, y, z = fruits

Outputting Variables: print()

Use print() to output variables. Separate with commas or use the + operator.

Example: Outputting Variables

print("Python is awesome")
print("Python", "is", "awesome")
print("Python " + "is " + "awesome")
print(5 + 10)
print(5, "John") (Correct way to combine different types)

Global Variables

Global variables are accessible outside and inside functions.

Example: Global Variables

x = "awesome"; def myfunc(): print("Python is " + x); myfunc()

The global Keyword

Use global to create or modify global variables inside functions.

Example: Using global

def myfunc(): global x; x = "fantastic"; myfunc(); print("Python is " + x)

Built-in Data Types

Python has various built-in data types, including: str, int, float, complex, list, tuple, dict, set, bool, and more.

Getting the Data Type: type()

Use type() to find the data type of a variable.

x = 5;
print(type(x))

Setting Data Types: Implicitly

Data type is set when you assign a value.

x = "Hello";
x = 20;
x = 20.5;
x = 1j;
x = [1,2,3];
x = (1,2,3);
x = {"a":1};
x = {1,2,3};
x = True;
x = b"Hello";
x = bytearray(5);
x = memoryview(bytes(5));
x = None

Setting Data Types: Explicitly

Use constructor functions to specify data types.

x = str("Hello");
x = int(20);
x = float(20.5);
x = complex(1j);
x = list((1,2,3));
x = tuple((1,2,3));
x = dict(a=1);
x = set((1,2,3));
x = bool(5);
x = bytes(5);
x = bytearray(5);
x = memoryview(bytes(5))

Python Numbers: int, float, complex

Python has three numeric types: int, float, and complex.

Example: Numeric Types

x = 1;

print(type(z))

Type Conversion

Convert numeric types using int(), float(), complex().

x = 1;
y = 2.8;
z = 1j;
a = float(x);
b = int(y);
c = complex(x);
print(a, b, c)

Random Numbers

Use the random module to generate random numbers.

import random;
print(random.randrange(1, 10))

Module 4: Python Strings

Strings

Strings are surrounded by single or double quotes.
'hello' is the same as "hello".

print("Hello")
print('Hello')

Assigning Strings to Variables

Assign strings with the variable name, equals sign, and the string.

a = "Hello"
print(a)

Multiline Strings

Use three quotes for multiline strings.

a = """Lorem ipsum dolor sit amet,
consectetur adipiscing elit,
sed do eiusmod tempor incididunt
ut labore et dolore magna aliqua."""
print(a)

a = '''Lorem ipsum dolor sit amet,
consectetur adipiscing elit,
sed do eiusmod tempor incididunt
ut labore et dolore magna aliqua.'''
print(a)

Strings as Arrays

Strings are arrays of bytes representing Unicode characters. Use square brackets to access elements.

a = "Hello, World!"
print(a[1])

Looping Through a String

Loop through characters with a for loop.

for x in "banana":
print(x)

String Length

Use len() to get string length.

a = "Hello, World!"
print(len(a))

Check String: in

Use in to check if a phrase is present.

txt = "The best things in life are free!"
print("free" in txt)

if "free" in txt:
print("Yes, 'free' is present.")

Check String: not in

Use not in to check if a phrase is NOT present.

txt = "The best things in life are free!"
print("expensive" not in txt)

if "expensive" not in txt:
print("No, 'expensive' is NOT present.")

Slicing

Use slice syntax to return a range of characters.

Module 6: Modify Strings

Upper Case

The upper() method returns the string in upper case.

a = "Hello, World!"
print(a.upper())

Lower Case

The lower() method returns the string in lower case.

a = "Hello, World!"

Remove Whitespace

The strip() method removes whitespace from the beginning or end.

a = " Hello, World! "
print(a.strip()) # returns "Hello, World!"

Replace String

The replace() method replaces a string with another string.

a = "Hello, World!"
print(a.replace("H", "J"))

Split String

The split() method splits the string into substrings based on a separator.

a = "Hello, World!"
print(a.split(",")) # returns ['Hello', ' World!']

Module 8: Python Booleans

Boolean Values

Booleans represent True or False. You can evaluate expressions to get Boolean answers.

print(10 > 9)
print(10 == 9)
print(10 < 9)

Booleans in If Statements

Booleans are used to control the flow of if statements.

a = 200
b = 33
if b > a:
print("b is greater than a")
else:
print("b is not greater than a")

Evaluate Values and Variables: bool()

Use bool() to evaluate any value and get True or False.

print(bool("Hello"))
print(bool(15))

x = "Hello"
y = 15
print(bool(x))
print(bool(y))

Most Values are True

Any string, except "", any number except 0, and any list, tuple, set, dict except empty ones, are True.

bool("abc")
bool(123)
bool(["apple", "cherry", "banana"])

Some Values are False

False, None, 0, "", (), [], {}, and objects with __len__() returning 0 or False are False.

bool(False)
bool(None)
bool(0)
bool("")
bool(())
bool([])
bool({})

class myclass():
def __len__(self):
return 0
myobj = myclass()
print(bool(myobj))

Functions Returning Booleans

Functions can return Boolean values.

def myFunction():
return True
print(myFunction())

if myFunction():
print("YES!")
else:
print("NO!")

isinstance() Function

isinstance() checks if an object is of a certain data type.

x = 200
print(isinstance(x, int))

Module 10: Python Lists

Lists

Lists store multiple items in a single variable, created using square brackets.

thislist = ["apple", "banana", "cherry"]
print(thislist)

List Items

Ordered, changeable, allow duplicates, indexed (first item [0]).

thislist = ["apple", "banana", "cherry", "apple", "cherry"]
print(thislist)

List Length

Use len() to get the number of items.

thislist = ["apple", "banana", "cherry"]
print(len(thislist))

List Items - Data Types

Lists can contain any data type (string, int, boolean) or mixed types.

list1 = ["apple", "banana", "cherry"]
list2 = [1, 5, 7, 9, 3]
list3 = [True, False, False]
list1 = ["abc", 34, True, 40, "male"]

type()

Lists are objects with the data type 'list'.

mylist = ["apple", "banana", "cherry"]
print(type(mylist))

The list() Constructor

Use list() to create a new list.

thislist = list(("apple", "banana", "cherry"))
print(thislist)

Access List Items

Access items by index, negative index, or range.

thislist = ["apple", "banana", "cherry"]
print(thislist[1])
print(thislist[-1])
thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]

print(thislist[2:])
print(thislist[-4:-1])

Check if Item Exists

Use in to check if an item is present.

thislist = ["apple", "banana", "cherry"]
if "apple" in thislist:
print("Yes, 'apple' is in the fruits list")

Change List Items

Change item values by index or range.

thislist = ["apple", "banana", "cherry"]
thislist[1] = "blackcurrant"

thislist[1:3] = ["watermelon"]
print(thislist)

Insert Items

Use insert() to add items at a specified index.

thislist = ["apple", "banana", "cherry"]
thislist.insert(2, "watermelon")
print(thislist)

Append Items

Use append() to add items to the end.

thislist = ["apple", "banana", "cherry"]
thislist.append("orange")
print(thislist)

Extend List

Use extend() to add elements from another list or iterable.

thislist = ["apple", "banana", "cherry"]

thislist.extend(tropical)

print(thislist)

del thislist[0]

thislist = ["apple", "banana", "cherry"]
del thislist
thislist = ["apple", "banana", "cherry"]
thislist.clear()
print(thislist)

Module 11: List Operations

List Comprehension

Shorter syntax for creating a new list from an existing list.

fruits = ["apple", "banana", "cherry", "kiwi", "mango"]
newlist = [x for x in fruits if "a" in x]
print(newlist)

Syntax

newlist = [expression for item in iterable if condition == True]

Condition

Filter items based on a condition.

newlist = [x for x in fruits if x != "apple"]
newlist = [x for x in fruits]

Iterable

Use any iterable object (list, tuple, range, etc.).

newlist = [x for x in range(10)]
newlist = [x for x in range(10) if x < 5]

Expression

Manipulate items before adding to the new list.

newlist = [x.upper() for x in fruits]
newlist = ['hello' for x in fruits]
newlist = [x if x != "banana" else "orange" for x in fruits]

Sort Lists

Use sort() to sort lists alphanumerically or numerically.

thislist = ["orange", "mango", "kiwi", "pineapple", "banana"]
thislist.sort()
print(thislist)
thislist = [100, 50, 65, 82, 23]
thislist.sort()
print(thislist)

Sort Descending

Use reverse = True to sort in descending order.

thislist = ["orange", "mango", "kiwi", "pineapple", "banana"]
thislist.sort(reverse=True)
print(thislist)
thislist = [100, 50, 65, 82, 23]
thislist.sort(reverse=True)
print(thislist)

Customize Sort Function

Use key = function to customize sorting.

def myfunc(n):
return abs(n - 50)
thislist = [100, 50, 65, 82, 23]
thislist.sort(key=myfunc)
print(thislist)

Case Insensitive Sort

Use key = str.lower for case-insensitive sorting.

thislist = ["banana", "Orange", "Kiwi", "cherry"]
thislist.sort(key=str.lower)
print(thislist)

Reverse Order

Use reverse() to reverse the order of elements.

thislist = ["banana", "Orange", "Kiwi", "cherry"]
thislist.reverse()
print(thislist)

Copy Lists

Use copy() or list() to create a copy of a list.

thislist = ["apple", "banana", "cherry"]
mylist = thislist.copy()
print(mylist)
thislist = ["apple", "banana", "cherry"]
mylist = list(thislist)
print(mylist)

Join Lists

Use +, append(), or extend() to join lists.

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]
list3 = list1 + list2
print(list3)
list1 = ["a", "b", "c"]
list2 = [1, 2, 3]
for x in list2:
list1.append(x)
print(list1)
list1 = ["a", "b", "c"]
list2 = [1, 2, 3]
list1.extend(list2)
print(list1)

Tuples

Tuples store multiple items in a single variable, ordered and unchangeable, written with round brackets.

thistuple = ("apple", "banana", "cherry")
print(thistuple)

Tuple Items

Ordered, unchangeable, allow duplicates, indexed (first item [0]).

thistuple = ("apple", "banana", "cherry", "apple", "cherry")
print(thistuple)

Tuple Length

Use len() to get the number of items.

thistuple = ("apple", "banana", "cherry")
print(len(thistuple))

Create Tuple With One Item

Add a comma after the item.

thistuple = ("apple",)
print(type(thistuple))
thistuple = ("apple")
print(type(thistuple))

Tuple Items - Data Types

Tuples can contain any data type (string, int, boolean) or mixed types.

tuple1 = ("apple", "banana", "cherry")
tuple2 = (1, 5, 7, 9, 3)
tuple3 = (True, False, False)
tuple1 = ("abc", 34, True, 40, "male")

type()

Tuples are objects with the data type 'tuple'.

mytuple = ("apple", "banana", "cherry")
print(type(mytuple))

The tuple() Constructor

Use tuple() to create a new tuple.

thistuple = tuple(("apple", "banana", "cherry"))
print(thistuple)

Access Tuple Items

Access items by index, negative index, or range.

thistuple = ("apple", "banana", "cherry")
print(thistuple[1])
print(thistuple[-1])
thistuple = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(thistuple[2:5])
print(thistuple[:4])
print(thistuple[2:])
print(thistuple[-4:-1])

Check if Item Exists

Use in to check if an item is present.

thistuple = ("apple", "banana", "cherry")
if "apple" in thistuple:
print("Yes, 'apple' is in the fruits tuple")

Update Tuples

Convert tuple to list, change, and convert back.

x = ("apple", "banana", "cherry")
y = list(x)
y[1] = "kiwi"
x = tuple(y)
print(x)

Add Items

Convert to list, append, and convert back, or add tuple to tuple.

thistuple = ("apple", "banana", "cherry")
y = list(thistuple)
y.append("orange")
thistuple = tuple(y)
thistuple = ("apple", "banana", "cherry")
y = ("orange",)
thistuple += y
print(thistuple)

Remove Items

Convert to list, remove, and convert back, or delete the tuple.

thistuple = ("apple", "banana", "cherry")
y = list(thistuple)
y.remove("apple")
thistuple = tuple(y)
thistuple = ("apple", "banana", "cherry")

Unpack Tuples

Extract values back into variables.

fruits = ("apple", "banana", "cherry")
(green, yellow, red) = fruits
print(green)
print(yellow)
print(red)

Using Asterisk*

(green, *tropic, red) = fruits

Loop through tuples using for loops, index numbers, or while loops.

print(x)

for i in range(len(thistuple)):
print(thistuple[i])
thistuple = ("apple", "banana", "cherry")
i = 0
while i < len(thistuple):
print(thistuple[i])
i = i + 1

print(f.readline())
f.close()

Write to a File

Use "a" to append or "w" to overwrite content.

f = open("demofile2.txt", "a")
f.write("Now the file has more content!")
f.close()

Create a File

Use "x", "a", or "w" to create a new file.

f = open("myfile.txt", "x")

Delete a File

Import the os module and use os.remove() to delete a file.

import os
os.remove("demofile.txt")

Delete a Folder

Use os.rmdir() to delete an empty folder.

import os
os.rmdir("myfolder")
